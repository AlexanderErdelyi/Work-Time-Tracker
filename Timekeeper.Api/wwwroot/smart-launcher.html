❓ Help & Troubleshooting

🎯 Quick Start

  1. Download the Smart Launcher from the Scripts tab
  2. Right-click the downloaded .ps1 file → Run with PowerShell
  3. Browser opens automatically to Timekeeper!

🔧 Port Already in Use?

  1. Go to API Ports or Frontend Ports tab
  2. Scan shows which ports are available (green) vs in use (red)
  3. Select an available port
  4. Download the start script for that specific port

🚫 Need to Stop Services?

  1. Download Kill Port Process to stop a specific port
  2. Or download Stop All Services to stop everything
  3. Close the PowerShell windows that are running the services
Manual Port Check (PowerShell)
Get-NetTCPConnection -LocalPort 5000-5020 | Select-Object LocalPort, State, OwningProcess | Format-Table
Find Process Using Port (PowerShell)
$port = 5000
$conn = Get-NetTCPConnection -LocalPort $port -ErrorAction SilentlyContinue
if ($conn) { Get-Process -Id $conn.OwningProcess }
