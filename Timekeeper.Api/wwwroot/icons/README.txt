🎨 TIMEKEEPER PWA ICONS
========================

Modern, professional icons with gradient background (indigo to blue) and polished clock design.

📥 HOW TO GENERATE:
1. Open: http://localhost:5174/generate-icons.html
2. Click "Download All Icons" button
3. Save all 4 icons to this folder

📦 REQUIRED ICONS:
✓ icon-192x192.png - Standard app icon (Android, Chrome)
✓ icon-512x512.png - High-quality app icon (install splash)
✓ icon-192x192-maskable.png - Adaptive icon (Android safe zone)
✓ icon-512x512-maskable.png - High-quality adaptive icon

🎨 DESIGN FEATURES:
- Modern gradient: Indigo → Blue
- Rounded square shape (18% corner radius)
- White clock face with tick marks
- 10:10 time display (classic watch marketing time)
- Subtle shadows and depth
- Professional polish

💡 CUSTOMIZATION:
Edit generate-icons.html to change:
- Colors (gradient stops)
- Clock time (hand angles)
- Border radius
- Shadow effects

The new icons match modern app design standards and look great on both light and dark backgrounds!
