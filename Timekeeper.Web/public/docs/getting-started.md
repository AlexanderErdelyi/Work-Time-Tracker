# Getting Started

## 1. Sign in

Choose your preferred sign-in method:

- Email + password
- Microsoft
- GitHub
- Windows credentials / integrated auth (if enabled)

## 2. Check in

Before tracking time, use **Check In** in the top bar.

## 3. Start your first timer

Go to **Dashboard** and start a timer for a task.

## 4. Stop and review

Stop the timer, then verify details in **Time Entries**.

## 5. Daily closeout

- Submit entries (if your process requires approval)
- Check your workday and breaks
- Review reports
