# FAQ

## Why can I not start a timer?

Usually this is due to missing task selection, auth/session issues, or API connectivity.

## What is the difference between Work Days and Time Entries?

- Work Days: attendance/check-in/check-out and break context
- Time Entries: task-level tracked work records

## Can I edit submitted entries?

Depends on your role and approval workflow state.

## Where should I report issues?

Use your team support process and include screenshots plus timestamps.
